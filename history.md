### 插件安装
* 路由
```
npm install router
```
* vuex
```
npm install vuex
```
* less
```
npm install less less-loader
```
* axios
```
npm install axios vue-axios
```
* vue-cli-plugin-style-resources-loader
```
vue add style-resources-loader
```
* mockjs
```
npm install mockjs
```
